# Изобразительное искусство

## Упражнения

### Расширение поля зрения

Идти по улице, смотреть вперёд и читать вывески боковым зрением.

### Практика рисования

Нарисовать:
- Пятиконечную звезду
- Окружность
- Спираль
- Квадрат из точек
- Квадрат из линий (в разные направления)
